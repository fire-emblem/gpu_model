Perfetto waitcnt slots demo guide

Use timeline.perfetto.json for timeline inspection, grep-friendly structure, and metadata.

Recommended order:
1. cycle/timeline_gap
   Best for obvious blank bubbles after buffer_load_dword.
   Look for: long empty gap, stall_waitcnt_global, load_arrive, then resumed instruction.

2. cycle/same_peu_slots
   Best for resident-fixed slot view across multiple PEUs.
   Look for: D0/A0/P0..P3, each with S0..S3, wave_launch, wave_switch_away,
   stall_waitcnt_global, load_arrive, wave_exit.

3. cycle/switch_away_heavy
   Best for scheduler rotation and repeated switch-away visibility.
   Look for: dense wave_switch_away markers with little waitcnt involvement.

4. st/same_peu_slots and mt/same_peu_slots
   Best for logical_unbounded slot behavior plus dense wave_switch_away markers.
   Look for: P0 with many S* tracks, wave_switch_away, and wave_exit on
   logical_unbounded slots.

Key meanings:
- wave_launch: wave starts on that slot track
- wave_switch_away: wave is scheduled away from execution
- stall_waitcnt_global: wave is blocked waiting for global memory
- load_arrive: memory return arrives
- wave_exit: wave finishes
- empty timeline span with no instruction slices: visible bubble
